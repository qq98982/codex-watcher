## 2026-01-28: Session 过滤功能

**类型**：新功能

**背景**：
需要隐藏插件相关的中间会话，避免干扰用户界面和搜索结果。

**改动**：
- 添加 `SessionFilter` 函数类型到 search 包
- 实现通过 API 层设置的过滤逻辑

**影响**：
- 影响：API 层 (`routes.go`) 和 search 包
- 过滤器应用于主 UI、/api/sessions 和搜索结果

**相关文件**：
- `internal/api/routes.go`
- `internal/search/search.go`
