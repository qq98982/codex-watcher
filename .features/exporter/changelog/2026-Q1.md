## 2026-01-28: 初始版本

**类型**：新功能

**背景**：
创建数据导出功能，支持多种格式和过滤策略。

**改动**：
- 实现 `WriteSession()` 支持单会话导出
- 实现 `WriteByDirFlat()` 支持目录扁平导出
- 实现 `WriteByDirAllMarkdown()` 支持完整 Markdown 导出
- 支持 Filters 结构控制导出策略

**影响**：
- 用户可以导出会话数据
- 支持多种格式和模式

**相关文件**：
- `internal/exporter/exporter.go`
