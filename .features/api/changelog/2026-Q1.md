## 2026-01-28: Session 过滤功能

**类型**：新功能

**背景**：
需要隐藏插件相关的中间会话（如 thedotmack），避免干扰用户界面。

**改动**：
- 添加 `shouldHideSession()` 过滤函数
- 在 `AttachRoutes()` 中设置 `search.SessionFilter`
- 过滤器应用到主 UI、/api/sessions 端点

**影响**：
- 插件中间会话不再显示在 UI 和搜索结果中
- 会话仍被索引，只是被过滤隐藏

**相关文件**：
- `internal/api/routes.go`
- `internal/search/search.go`

**Commit/PR**：cd99ad6
