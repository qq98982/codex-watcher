## 2026-01-28: Session 过滤支持

**类型**：新功能

**背景**：
需要支持在搜索时过滤特定会话（如插件中间会话）。

**改动**：
- 添加 `SessionFilter` 全局变量（函数类型）
- 在 `Exec()` 中应用过滤器到 sessions 列表

**影响**：
- 搜索结果可按需过滤会话
- 过滤逻辑由 API 层设置

**相关文件**：
- `internal/search/search.go`
- `internal/api/routes.go`

**Commit/PR**：cd99ad6
